# XCX Language Support

This extension provides rich language support for the XCX programming language.

## Features

- **Syntax Highlighting**: Comprehensive highlighting for all XCX keywords, types, operators, and literals.
- **Snippets**: Handy snippets for common language constructs (functions, loops, tables, etc.).
- **Language Configuration**: Support for comments, brackets, and auto-closing pairs.

## Syntax Highlighting

The extension highlights:
- Keywords: `if`, `then`, `else`, `while`, `for`, `func`, `include`, etc.
- Types: `i`, `f`, `s`, `b`, `array`, `set`, `map`, `table`, `json`, `date`.
- Built-ins: `halt`, `alert`, `error`, `fatal`, `store`, `random`.
- Operators: Arithmetic, comparison, logical, and special XCX operators like `>!`, `>?`, `->`, `++`.
- Literals: Integers, floats, strings, and multi-line raw blocks (`<<< >>>`).

## Installation

1. Download the `.vsix` file.
2. Open VS Code.
3. Go to the Extensions view.
4. Click on the "..." menu and select "Install from VSIX...".
5. Choose the downloaded `.vsix` file.
